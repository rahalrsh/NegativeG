Negative-G

For the first time humans have discovered a planet with negative gravity. This negative gravity will push you away from the planet's surface. Use your special jet pack to survive in this planet.



How to play

Double click the Negative-G.jar to play the game



Game Controls

DOWN arrow key to activate jet pack
LEFT arrow key to move left
RIGHT arrow key to move right
SPACE to restart game
